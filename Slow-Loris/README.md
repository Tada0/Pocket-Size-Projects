Slow Loris Attacker
=======================

#### Usage: 
```
python3 app.py [IP] [N] 
```

#### eg. Usage: 
```
python3 app.py 192.168.1.1 300
```

Creates 300 connections with 192.168.1.1

Each socket sends 1 byte every 15s

Works on Apache, Flask etc.
